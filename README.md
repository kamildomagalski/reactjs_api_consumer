# ReactJS API consumer app 
> If you need to find an interesting movie for tonight, that's an app for you 

## Description
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).</br>

This app provides basic functionality for one api call only. </br>
App can be developed by adding new components and more routes, introduce global state e.g. Redux/Redux-Thunk. </br>
Some basic components can be also easily reused thanks to it's structure.</br>

This version of an app contains only desktop version withot RWD.

## Setup

### `npm install`
 To start you have to install locally all packages using npm.
### Provide your own API KEY 
Go to the [TheMovieDatabase](https://developers.themoviedb.org/3/getting-started/introduction),
sign in/login to your account and generate api key.
When done, simply create .env file in root folder and place the key there.

### `npm start`
Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
